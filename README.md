# E-Commerce Database Management System

Overview
This project is a SQL Server-based relational database designed to simulate a real-world e-commerce platform. It demonstrates database design principles, relationships, constraints, and SQL programming techniques.

 Database Schema

The database consists of the following tables:

- Customers
- Orders
- OrderDetails
- Products
- Categories
- Suppliers
- Reviews
- Employees
- Shippers
- Payments

Features

- Relational Database Design
- Primary & Foreign Keys
- Data Integrity Constraints
- SQL Queries
- Joins
- Aggregate Functions
- Views
- Stored Procedures
- Triggers
- Transactions

Technologies

- Microsoft SQL Server
- T-SQL

 Project Purpose
